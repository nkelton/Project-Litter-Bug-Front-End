Calculation

Gigabytes Generated

As Project Litter Bug randomly generates useless content and uploads it to YouTube, one of the things it does is keeps track of the byte size of each file it uploads. The size of every piece of content is then summed together and converted from bytes to gigabytes, which is where the “Gigabytes Generated” value comes from on the home page.

= (Bytes) * 109

Energy Used

In order to perform this calculation, we must take the size of each piece of content and multiply it by the number of hours since it was uploaded. The resulting value constitutes the "total weight" of that video. This value is then once again converted into gigabytes and the aggregate of all videos is summed. Finally, the summed value is multiplied by the constant 5.12 kWh per GB. This constant is the average amount of power the Internet uses to support the utilization of every GB of data. It is taken from a study by the ACEEE, The Megawatts behind Your Megabytes: Going from Data-Center to Desktop.

= ((Content Byte Size * Time Since Upload) * 109) * 5.12

Total Cost

Likewise, the cost is derived using another constant from the aforementioned study. It assumes an average electricity cost of $0.10 per kWh which equates to a system energy cost of $0.51 per GB. This constant is again multiplied by the number of gigabytes the script has generated overtime, and results in the value displayed on the homepage.

= (Gigabytes) * 0.51

Total CO2 Emissions

This value is calculated by multiplying the "Energy Used" value by the Emissions Factor defined for the EPA's greenhouse gas calculator. The resulting unit is CO2 by Metric Tons.

Emissions Factor = 1,559 lbs CO2/MWh × (4.536 × 10-4 metric tons/lb) × 0.001 MWh/kWh = 7.07 × 10-4 metric tons CO2/kWh

= Energy Used * 7.07 × 10-4

Notes

  • Barring the number of gigabytes generated, the values displayed on the homepage should be looked at as raw estimations and subject to error. The point of these values are not to be scientifically accurate but to help convey the idea that the data a user uploads (or accesses) on the Internet has a calculable impact on the world’s resources.
  • This calculation does not take into account the amount of energy required to generate the random content, stream the videos on YouTube, or run the server this website is hosted on.
